Introduction to HTML by Valerie Dollery

Doctype

the Doctype informs the browser what type of document to expect

example of the code

<!doctype html>

html

HTML is the standard markup language for creating Web pages

<html></html>

head

This element is a container for metadata (data about data) and is placed between the html tag and the body tag

<head></head>

body

This element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.There can only be one body element in an HTML document

<body></body>

title

This tag defines the title of the document. The title must be text-only, and it is shown in the browser's title bar or in the page's tab.

<title></title>

p

The HTML element defines a paragraph.A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

working example

<p>This is a paragraph</p>

This is a paragraph

span

This tag is an inline container used to mark up a part of a text, or a part of a document.This tag is easily styled by CSS. The tag is much like the div element, but div is a block-level element and span is an inline element.

My name is<span>Valerie</span>

working example

My name is Valerie

div

This tag defines a division or a section in an HTML document

Lorem Ipsum
I am a div
dolor sit amet.

working example

Lorem Ipsum
I am a div
dolor sit amet.

Formatting Elements - strong emphasis

The strong tag is used to define text with strong importance. The content inside is typically displayed in bold. The em tag is used to define emphasized text. The content inside is typically displayed in italic

<p> <strong> - Important text </strong> <em> - Emphasized text </em> </p>

working example

- Important text - Emphasized text

tables & caption

HTML tables allow web developers to arrange data into rows and columns.The caption tag defines a table caption,the caption tag must be inserted immediately after the table tag.

<table> <caption> Email information </caption> <tr> <td>Emil</td> <td>Tobias</td> <td>Linus</td> </tr> </table>

working example

Email information
Emil Tobias Linus

lists

HTML lists allow web developers to group a set of related items in lists. An ordered list is a numbered list. An unordered list is a bulleted list and a definition list is a list of terms with a description of each.

<ul> <h2>Unordered List</h2> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ul></p>

<p><ol> <h2>Ordered List</h2> <li>Coffee</li> <li>Tea</li> <li>Milk</li> </ol></p>

<p><dl> <h2>Definition List</h2> <dt>Coffee</dt> <dd>- black hot drink</dd> <dt>Milk</dt> <dd>- white cold drink</dd> </dl></p>

working example

    Unordered List

  • Coffee
  • Tea
  • Milk

    Ordered List

  1. Coffee
  2. Tea
  3. Milk

Definition List

Coffee
- black hot drink
Milk
- white cold drink

Images & alt

Images can improve the design and the appearance of a web page. Alt - Specifies an alternate text for the image

<img src="images/alcatraz.jpg" width="400" height="266" alt=""/>

working example

links

The link tag defines the relationship between the current document and an external resource

<a href="https://www.w3schools.com/">Visit W3Schools.com!</a>

working example

Visit W3Schools.com!

quote & blockquote

The blockquote tag specifies a section that is quoted from another source and is usually indented.The HTML q tag defines a short quotation.

<blockquote cite="http://www.worldwildlife.org/who/index.html">
For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.
</blockquote>

<p>WWF's goal is to: <q>Build a future where people live in harmony with nature.</q></p>

working example

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

WWF's goal is to: Build a future where people live in harmony with nature.

Abbreviations

The abbr tag defines an abbreviation or an acronym, like "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

The <abbr title="World Health Organization">WHO</abbr> was founded in 1948.

working example

The WHO was founded in 1948.

Meta

The meta tag defines metadata about an HTML document. Metadata is data (information) about data. Metadata will not be displayed on the page, but is machine parsable.

<head> <meta charset="UTF-8"> <meta name="description" content="Free Web tutorials"> <meta name="keywords" content="HTML, CSS, JavaScript"> <meta name="author" content="John Doe"> <meta name="viewport" content="width=device-width, initial-scale=1.0"> </head>
